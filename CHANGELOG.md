# Changelog

All notable changes to DevLens will be documented in this file.

## 0.1.0-alpha.3

### Added

- Runtime error monitoring support.
- Unhandled promise rejection tracking.
- Unified error count in DevLens bar.
- Render count in floating debugbar.
- NPM package README files for React and Next.js packages.

### Improved

- Network details panel readability.
- Responsive drawer tabs.
- Mobile floating bar layout.
- Empty state UI.
- Runtime monitor lifecycle safety.
- Workspace dependency publishing flow.

### Fixed

- Prevented duplicate monitor installation during remounts and hot reload.
- Fixed npm package dependency publishing by using workspace-aware pnpm publishing.
- Fixed corrupted drawer close button character.

## 0.1.0-alpha.2

### Fixed

- Added package README files for npm package pages.

## 0.1.0-alpha.1

### Added

- Initial alpha release.
- React adapter.
- Next.js adapter.
- DevLens UI package.
- Core runtime package.
- Shared utilities package.
